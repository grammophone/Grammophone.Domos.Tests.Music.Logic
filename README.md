# Grammophone.Domos.Tests.Music.Logic

Logic layer for the Domos music test application.

This project defines `MusicSession`, manager classes, role and disposition names, and the programmatic permissions setup used by the shared music test cases.

## Target Frameworks

- `net472`
- `netstandard2.1`
